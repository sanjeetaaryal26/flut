Project Overview

Bite Finder is a restaurant and food discovery application. This repository contains only the login and signup functionality implemented as part of an academic assignment. The complete application is not included.

Scope of Implementation

User signup functionality

User login functionality

Backend authentication API integration

Tech Stack

Frontend: Flutter

Backend: Node.js (Express.js)

Architecture: Clean Architecture

This project is limited to authentication features and serves as a foundation for future development of the Bite Finder application.
